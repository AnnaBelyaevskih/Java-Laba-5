## Задание 1. Шаблоны (Дробь)

### Интерфейс `FractionValue`

Интерфейс определяет контракт для работы с дробью:
- получение вещественного значения;
- установка числителя;
- установка знаменателя.

### Класс `Fraction`

Класс описывает математическую дробь и содержит:
- числитель и знаменатель;
- кэширование вещественного значения дроби;
- проверку корректности знаменателя (знаменатель всегда положительный);
- переопределённые методы `toString()`, `equals()` и `hashCode()`.

Кэш сбрасывается при изменении числителя или знаменателя.

## Задание 2. Структурные шаблоны (Мяуканье котов)

### Класс `Cat`

Класс представляет сущность «Кот»:
- хранит имя кота;
- содержит метод `meow()`, выводящий сообщение о мяуканье;
- может быть приведён к строке вида `кот: Имя`.

### Интерфейс `Meowable`

Интерфейс описывает всех объектов, способных мяукать.  
Содержит метод `meow()`.

### Класс `MeowCounterCat`

Адаптер для класса `Cat`, который:
- оборачивает объект `Cat`;
- считает количество вызовов мяуканья;
- реализует интерфейс `Meowable`.

Это позволяет определить количество мяуканий без изменения исходного класса `Cat`.

### Класс `MeowProcessor`

Содержит метод, принимающий набор объектов `Meowable` и вызывающий у каждого метод `meow()`.

## Задание 3. Список (`List`)

### Класс `ListTask1`

Реализует удаление всех вхождений заданного элемента из списка.

Для корректного удаления используется `Iterator`, что предотвращает ошибки во время обхода коллекции.

## Задание 4. Ассоциативный массив (`Map`)

### Класс `LoginGenerator`

Формирует уникальные логины учеников:
- при первом появлении фамилии логин совпадает с фамилией;
- при повторных появлениях к фамилии добавляется порядковый номер.

Для хранения данных используется `HashMap`.

## Задание 5. Множества (`Set`)

### Класс `VoicedConsonantsTask`

Класс анализирует текстовый файл и:
- определяет звонкие согласные буквы русского языка;
- находит буквы, встречающиеся более чем в одном слове;
- возвращает результат в отсортированном виде (`TreeSet`).

## Задание 6. Очередь (`Queue`)

### Класс `QueueReverseCopy`

Переписывает элементы из одной очереди в другую в обратном порядке.

Для этого используется вспомогательный стек. Исходная очередь при этом не изменяется.

## Задание 7. Stream API

### Класс `Point`

Описывает точку на плоскости:
- координаты `X` и `Y`;
- переопределённые методы `equals()` и `hashCode()` для корректной работы со Stream API.

### Класс `Polyline`

Описывает ломаную линию как последовательность точек.  

### Класс `StreamPointsTask`

Использует Stream API для:
- удаления точек с одинаковыми координатами;
- преобразования отрицательных координат `Y` в положительные;
- сортировки точек по координате `X`;
- формирования объекта `Polyline`.

### Класс `PeopleGroupingTask`

Обрабатывает строки формата `имя:номер` и:
- приводит имена к корректному регистру;
- удаляет записи без номера;
- группирует имена по номеру.

Результат возвращается в виде `Map<Integer, List<String>>`.

## Главный класс `Main`

Класс `Main`:
- выводит меню выбора задания;
- организует ввод данных с клавиатуры;
- проверяет корректность ввода;
- демонстрирует работу всех заданий.
